# Alphanumeria Specification

Alphanumeria is the foundational language system behind Numia.

It is a structured, human-readable encoding system designed to represent meaning through deterministic character substitution.

---

## Core Principle

Not encryption.  
Not code.  

A shared meaning system.

---

## Level 1 — Base Mapping

A = 1  
E = 2  
I = 3  
O = 4  
U = 5  
Y = 6  

---

## Examples

HELLO → H2LL4  
NUMIA → N5M31  
VALUE → V1L52  

---

## Design Intent

Level 1 is:
- simple
- deterministic
- universal

It serves as:
- onboarding layer
- shared reference
- foundation for future complexity

---

## Future Levels

- Level 2 — obfuscated encoding  
- Level 3 — hybrid encoding  
- Level 4 — shuffled transformations  

---

## Philosophy

Language first.  
Meaning second.  
Everything else follows.

---

Numia begins with language.
