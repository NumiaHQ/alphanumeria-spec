# Numia Roadmap

Numia is building a language-first digital ecosystem.

---

## Phase 1 — Foundation (Current)

- Define Alphanumeria specification
- Establish Level-1 mapping (A=1, E=2, I=3, O=4, U=5, Y=6)
- Create documentation and examples
- Build early community and awareness

Status: In progress

---

## Phase 2 — Identity Layer (ONID)

- Develop ONID (Own Numia Identity)
- Encode names, brands, and identifiers
- Explore on-chain verification for authenticity
- Enable brand/customer anti-counterfeit use cases

---

## Phase 3 — Developer Adoption

- Provide SDK / libraries for encoding/decoding
- Build simple APIs
- Enable integration into apps and platforms
- Encourage developer contributions

---

## Phase 4 — Real-World Use Cases

- Domain alternatives (e.g. n5m31.com)
- Password systems
- Messaging layers
- Identity verification tools

---

## Phase 5 — Ecosystem Expansion

- Partnerships with platforms and brands
- Potential integrations with cybersecurity tools
- Expansion into AI / data labeling (Numia Labs)

---

## Philosophy

Numia does not start with hype.

It starts with structure.

Language → Philosophy → Token

Meaning precedes value.
